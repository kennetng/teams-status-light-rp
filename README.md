# teams-status-light-rp

## Motivation

This project check Teams user availability and activity to show which led to light up.

## Tech/framework

- Express
- Typescript
- onoff (For RP led)
- Node
- Msal-node (Microsoft auth)

## Installation

```
yarn
```

## Run application local

```
yarn dev
```

### Build application

```
yarn build
```
